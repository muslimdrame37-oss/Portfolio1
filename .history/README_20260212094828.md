# Portfolio1

Portfolio statique (HTML + Tailwind CSS) avec deux pages :
- `index.html` (projets + contact)
- `a_propos.html` (bio + compétences + expérience + contact)

## Lancer le projet (démo)

Pré-requis : Node.js installé.

1) Installer les dépendances
```bash
npm install
```

2) Générer le CSS Tailwind
```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css
```

3) Lancer un serveur local (recommandé)
```bash
npx http-server ./src -p 5173 -c-1
```

Ouvrir ensuite :
- http://localhost:5173/index.html
- http://localhost:5173/a_propos.html

### Mode "watch" (pendant que tu codes)
```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```
